1. 데이터 설명 및 보기

데이터 출처 및 설명

countries of the world.csv 데이터
countries of the world.csv는 227개의 행과 20개의 열로 구성되어 있으며, 각각의 열은 인구, 지역, 면적, 영아 사망률 등에 대한 정보가 들어 있다. 해당 데이터의 원본은 미국 정부의 데이터로 구성되며, 이는 공개 도메인으로 무료로 제공된다. 이 데이터를 Kaggle의 데이터 세트에서 정리하였으며, 공공 데이터로 제공된다. 본 데이터의 다양한 열을 활용하여 국가별 GDP를 예측 하는 머신러닝 모델을 구축하거나, 출생률과 사망률 데이터를 통한 인구 변화 동향을 예측하는데 주로 사용된다.

원본데이터 출처
https://www.cia.gov/library/publications/the-world-factbook/docs/faqs.html

countries of the world 데이터 출처
https://www.kaggle.com/fernandol/countries-of-the-world



List of Countries by number of Internet Users - Sheet1.csv 데이터
List of Countries by number of Internet Users - Sheet1.csv 데이터는 215개의 행과 6개의 열로 구성되어 있다. 본 데이터는 2018년 인터넷 사용자 수에 따른 국가 목록 데이터이다. 원본 데이터 수집은 Wikipedia에서 하였으며, 이를 csv 파일로 변환한 Kaggle의 오픈 데이터이다. 데이터에는 나라 이름, 인터넷 사용자 수, 인구 수, 인터넷 사용자 비율 등의 열이 있다. 본 데이터를 활용하여 인터넷을 사용하지 않은 인구가 많은 나라를 시장 타깃으로 삼는 데이터 분석이나, 인터넷 사용자 수를 시각화하는 분석에 사용되었다.

Life_expectancy_dataset.csv 데이터
Life_expectancy_dataset.csv 데이터는 223개의 행과 6개의 열로 구성되어 있다. 본 데이터는 국가별 남성의 기대수명과 여성의 기대수명, 그리고 종합적인 기대수명에 대한 정보가 들어있다. 해당 데이터의 원본은 Wikipedia의 List of copuntries by life expectancy 검색을 통해 얻을 수 있으며, 이 데이터를 csv 파일로 변환하여 Kaggle의 오픈 데이터로 제공한다. 기대 수명 이외로 대륙 단위의 지역 구분 열이 존재하며, 이를 통하여 대륙별 기대 수명을 구할 수 있다. 본 데이터를 이용하여 특정 나라의 남성이 여성에 비해 유난히 기대 수명이 짧은 이유를 분석하는데 본 데이터가 활용 되었으며, 나라 이름으로 다른 데이터와 병합 하여 사용하는 경우가 많다.

Life_expectancy_dataset 데이터 출처

분석 내용 설명

본 데이터 분석에서는 인터넷 사용률이 얼마나 의료 인프라에 영향을 주어 기대수명을 증가 시키고, 영아 사망률을 낮출 수 있는지 알아본다. 이를 위하여 국가의 다양한 정보가 담겨있고 GDP와 1000명의 출생 당 영아 사망률이 담겨 있는 countries of the world 데이터와 각 나라별 인터넷 사용률이 있는 List of Countries by number of Internet Users 데이터, 그리고 각 나라별 기대 수명이 있는 Life_expectancy_dataset을 나라 이름으로 병합하여 사용한다. GDP 증가에 따라서 의료 인프라에 영향을 줄 것으로 판단되므로, GDP와 인터넷 사용률의 상관 관계를 분석하며, GDP 증가에 따른 의료 인프라, 인터넷 사용률에 따른 의료 인프라 경향을 각각 분석한다.

4. 데이터 시각화

본 데이터 분석에서는 해당 나라의 인터넷 사용률과 같은 인터넷 인프라가 의료 서비스 제공에 있어서 얼마나 영향을 주는지에 대한 분석을 목표로 한다. 이를 위해서 의료 인프라에 영향을 줄 수 있는 또 다른 지표 중 하나인 GDP와의 상관 분석을 먼저 실시해본다.

GDP와 인터넷 사용자 비율의 상관 분석을 통하여 두 변수의 관계를 시각화하였다. 이를 통하여 다음과 같은 정보를 알 수 있다.

  • GDP가 증가할수록 인터넷 사용자 비율이 증가하는 경향을 보인다.
  • GDP가 적을 경우 GDP가 증가하면 인터넷 사용률이 급 속도로 증가하며, 일정 GDP가 넘어서부터는 인터넷 사용률의 성장이 둔화되는 것을 보인다.
  • GDP와 인터넷 사용률이 낮은 그룹에는 아프리카 지역이 몰려있다.
  • 유럽은 GDP가 높아도 인터넷 사용률이 매우 낮은 경우가 다수 존재한다.
  • 아시아와 중동 지역은 유럽에 비해 같은 GDP일 경우 인터넷 사용률이 높은 경향을 보인다.

GDP와 인터넷 사용률을 상관 분석한 결과에서 GDP가 증가할 수록 인터넷 사용률도 증가하는 경향을 보인다. 이를 확실히 보기 위하여 GDP와 인터넷 사용률을 그룹화하여 해당 나라의 GDP 그룹이 정해졌을 때, 어떠한 인터넷 사용률 그룹에 속할 확률이 높은지 빈도 분석을 실시한다.

describe()를 통하여 분석한 결과, 그룹을 min-25%, 25%-50%, 50%-75%, 75%-max로 4개의 범주로 구분한다. 이를 통하여 각 범주마다 같은 수의 데이터가 그룹화된다.

GDP 그룹별로 해당 나라의 인터넷 사용률이 속한 그룹을 빈도분석을 통하여 시각화를 했다.

  • GDP가 매우 낮은 저소득 그룹에서 대부분의 나라가 인터넷 사용률 낮음 그룹에 속한다.
  • GDP가 매우 높은 초고소득 그룹에서 대부분의 나라는 인터넷 보급률이 매우 높은 그룹에 속한다.
  • GDP가 중위소득을 차지하는 그룹에서는 인터넷 보급률도 보통 그룹에 속할 경우가 가장 많다.
  • GDP가 고소득 그룹인 경우에도 인터넷 보급률이 중간인 경우가 가장 많다.

GDP와 인터넷 사용률을 데이터 개수를 기준으로 4개의 범주형으로 구분하였을 때, GDP와 인터넷 사용률 그룹 수준이 비슷할 경우가 가장 많은 것을 알 수 있다. 이를 통하여 GDP와 인터넷 사용률이 연관이 있다는 것을 알 수 있다.

그렇다면, 같이 증가하는 경향을 보이는 GDP와 인터넷 사용률 중에서 어떠한 변수가 그 나라의 기대 수명이나 영아 사망률과 같은 의료 인프라에 더 상관이 있는지 알아본다. 이를 위하여 GDP와 기대수명의 관계를 시각화하며, 마찬가지로 인터넷 사용률과 기대수명을 시각화한다.

인터넷 사용자 비율이 낮은 그룹에서는 GDP가 증가함에 따라 기대 수명이 급격하게 증가하는 경향을 보인다. 그리고 인터넷 사용자 비율이 보통, 중간, 높은 그룹에서는 GDP가 증가함에 따라 기대 수명이 인터넷 사용자 비율이 낮은 그룹만큼 급격하게 증가하지는 않지만, 증가하는 경향을 보인다. 또한, 비슷한 GDP를 가진 나라들을 비교하더라도 인터넷 사용자 비율이 높은 나라가 인터넷 사용자 비율이 낮은 나라보다 기대수명이 높은 경향을 보이지만, 일정 인터넷 사용자 비율을 이미 보유한 나라들 사이에서는 두 그룹간의 격차가 적어보인다. 이는 인터넷 사용자 비율이 낮은 나라들은 인터넷 인프라를 구축하여 의료 서비스를 제공하는 것만으로도 충분한 기대 수명의 증가와 같은 의료 인프라 구축에 큰 효과를 볼 수 있다고 추측 가능하다.

GDP가 저소득, 중소득, 고소득인 나라의 경우에는 인터넷 사용자 비율이 늘어날 수록 기대수명이 증가하는 경향을 보인다. 그러나 GDP가 매우 높다면 인터넷 사용자 비율이 더 이상 기대수명에 영향을 미치지 못하는 것을 알 수 있다. 그리고 같은 수준의 인터넷 사용자를 보유할 경우에는 GDP가 기대 수명에 큰 영향을 미치는 것을 알 수 있다. 이를 통하여 GDP가 높고, 이미 인터넷 사용자 비율을 보유한 상태의 나라라면 인터넷 사용자 비율의 증가보다는 다른 방법의 의료 인프라 구축이 더욱 효율적이라 추측된다.


5. 통계 또는 머신러닝

데이터 시각화를 통하여 GDP와 인터넷 사용자 비율이 의료 인프라 구축을 통한 기대 수명 향상에 얼마나 영향을 미칠 수 있는지를 알아봤다. 이를 좀 더 확장하여 의료 인프라 판단 기준의 하나라고 볼 수 있는 영아 사망률 데이터를 통하여 과연 인터넷 사용자 비율이 영아 사망률에도 영향을 미칠 수 있는지 알아보도록한다. 이를 위하여 가설을 수립한다.

가설 가설설명
H0(영가설) 인터넷 사용자 비율이 낮은 그룹과 인터넷 사용자 비율이 높은 그룹의 영아 사망률 평균은 다르지 않다.
H1(대립가설) 인터넷 사용자 비율이 낮은 그룹과 인터넷 사용자 비율이 높은 그룹의 영아 사망률 평균은 다르다.

이를 증명하기 위해 T-분석을 실시한다.

  • world 데이터를 인터넷사용자비율그룹으로 그룹화를 한다.
  • 인터넷 사용자 비율이 높은 그룹과 낮은 그룹의 영아 사망률을 T-분석 하여 P-value를 본다.

world_internet = world.groupby('인터넷사용자비율그룹')
In [40]:
low_internet = world_internet.get_group('G1낮음')
high_internet = world_internet.get_group('G4높음')
In [41]:
tTestResult = stats.ttest_ind(low_internet['영아사망률'], high_internet['영아사망률'])
print(tTestResult)

Ttest_indResult(statistic=13.890630206801383, pvalue=8.524917592763567e-24)

T-분석 결과 P-value가 약 8.52e-24를 보이며, 이는 유의수준 0.05 이하로 대립가설인 인터넷 사용자 비율이 낮은 그룹과 인터넷 사용자 비율이 높은 그룹의 영아 사망률 평균은 다르다가 채택된다.

그렇다면, 이러한 인터넷 사용자 비율이 영아 사망률에 얼마만큼의 영향을 미칠 수 있는지 상관계수를 heatmap()을 통한 시각화로 확인한다.

그렇다면, 두 연속형 변수인 인터넷 사용자 비율과 영아 사망률 데이터의 인과 관계를 추측하기 위해 선형회귀모형을 만든다. 이를 위한 두 변수를 다음과 같이 구분한다.

변수 변수명
X(독립변수) 인터넷사용지비율
Y(종속변수) 영아사망률

두 변수를 이용하여 회귀 모형을 구축한다.

구축된 회귀 모형의 intercept는 93.3648이며, 인터넷 사용자 비율은 -101.6182이다. 이룰 통하여 다음과 같은 회귀 모형이 구축된다

영아사망률 = -101.6182 * 인터넷사용자비율 + 93.3648

또한, R-squared는 0.603에 해당한다. 이는 추정한 선형 모형이 주어진 자료에 적합한 정도를 재는 척도이며, 본 회귀식은 본 데이터를 60.3% 설명이 가능하다는 의미이다. 이는 해당 회귀식이 데이터를 충분히 설명 가능한 수치이다.

P > |t|는 독립변수의 유의확률이며 0.05 이하의 값이 되어야 독립변수가 종속변수에 영향을 미치는 것을 유의미 하다고 본다. 본 회귀식에서는 0이므로 인터넷 사용자 비율은 영아 사망률에 유의미하게 영향을 미친다고 할 수 있다.

Durbin-Watson은 전차의 독립성을 확인할 수 있는 수치이다. 일반적으로 1.5 ~ 2.5 사이이면 독립으로 판단하고 회귀모형이 적합하다고 하는데, 본 회귀식은 1.816으로 잔차들이 자기상관을 갖지 않는 다는 의미이다.

6. 결론 및 본인 차별화 포인트

본 데이터 분석에서는 인터넷 사용자 비율을 인터넷 인프라로 가정하고, 인터넷 사용자 비율이 기대 수명과 영아 사망률 데이터에 어느 정도 영향을 미칠지 분석하여, 인터넷 사용자 비율이 의료 서비스 수준과 의료 인프라 수준에 미치는 영향을 분석하였다. 분석 결과 인터넷 사용자 비율이 매우 낮은 나라에서는 인터넷 인프라 구축이 매우 효과적인 의료 서비스 및 의료 인프라의 향상을 제공할 수 있을 것으로 기대된다. 또한 이미 GDP와 인터넷 사용자 비율이 높은 나라에서는 더 높은 인터넷 인프라가 의료 서비스의 향상을 늘리는데 큰 효과를 얻지는 못할 것으로 예상된다. 이러한 나라들은 의료 서비스 구축을 위해 인터넷 인프라를 제외한 그 외의 분야에 투자를 하는 것이 적절해 보인다. 인터넷 사용률이 증가 할수록 영아 사망률이 낮아진다는 점은 T-분석을 통하여 증명을 하였고, 이에 대해 인터넷 사영률과 영아 사망률에 대한 선형 회귀식을 도출 하였으며, 이 선형 회귀식을 평가한 결과 해당 선형식이 인터넷 사용자 비율과 영아 사망률의 관계를 훌륭하게 나타낸다고 평가된다.

본 데이터 분석을 위해 사용한 차별화 포인트는 다음과 같다.

  • 데이터 분석 전에 분석하고 싶은 내용을 미리 기술하고, 점점 세분화하는 방식으로 진행되도록 하였다.
  • 데이터 분석이 용이하도록 변수명을 한글, 그리고 괄호와 띄어쓰기를 제거하는 과정을 하였으며, 데이터 병합에 용이하도록 각 데이터 테이블의 key에 해당하는 변수를 통일하였다.
  • 데이터 병합 이전에 분석할 데이터를 정하고, 이용하지 않을 데이터를 제거하여 데이터 병합 이후에 적절한 양의 데이터를 유지하도록 하였다.
  • 범주로 묶기 어렵고, 데이터 분석에 용이하지 못한 타입으로 적혀있는 범주형 데이터를 한글, 그리고 데이터 치환 과정에서 같은 대륙은 범주 하나로 묶어 그룹의 갯수를 줄였다.
  • 데이터 클린징 과정에서 타당한 근거를 들어 데이터 결측치보다 데이터 이상치를 먼저 제거하였다.
  • 실제 나라 데이터에서 쉽게 이상치를 파악할 수 있는 인구, 인구밀도, GDP 데이터는 정렬을 통하여 최댓값, 최솟값을 구한 후 실제 데이터와 비교 분석하여 이상치인지와 데이터를 분석에 사용할 것인지 파악하였다.
  • describe()와 boxplot()은 연속형 변수의 이상치만을 볼 수 있으므로, 범주형 데이터의 이상치를 따로 분석하기 위해 value_counts() 메소드를 통하여 이상치를 관측하였다.
  • 기대수명과 영아 사망률이 인터넷 사용자 비율 뿐만이 아니라 GDP에 영향을 크게 받을 것을 고려하여 인터넷 사용자 비율과 GDP를 상관분석 하였으며, 각각의 경우를 나눠 기대수명 변화 경향을 시각화하였다.
  • 연속형 변수를 범주형 변수로 그룹화 하며 describe()를 통하여 그룹을 min-25%, 25%-50%, 50%-75%, 75%-max로 구분하여 각각의 범주에 같은 수의 데이터가 그룹화되도록 하였다.
  • 회귀모형을 구축하고, 구축한 회귀식과 실제 데이터를 시각화하여 회귀 모형 구축이 잘 되었는지 평가하였다.
